# Custom Font Assets

Place your custom wedding font files here. The project is pre-configured to look for the following files:

1. `Breathing.woff2` (Recommended)
2. `Breathing.woff`
3. `Breathing.ttf` (Added support)

Once these files are added to this folder, the website will automatically load them for page titles. The current styling uses a beautiful serif fallback font (`Playfair Display`) until these files are present.
